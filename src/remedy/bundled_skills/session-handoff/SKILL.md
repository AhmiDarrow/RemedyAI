# Session Handoff

## When to use
End of a session, "summarize for later", switch models/agents, or `/handoff`.

## Steps
1. State goal and what was accomplished.
2. List key decisions and constraints.
3. List modified files and remaining work (ordered).
4. Note blockers and how to verify.
5. Persist via memory/handoff tools if available; otherwise write a clear markdown summary.
