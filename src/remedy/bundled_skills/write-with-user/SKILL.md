# Write With User

## When to use
Emails, docs, posts, RFCs, messages — especially when voice/tone matters.

## Steps
1. Check profile facts for tone preferences (formal/casual, short/long).
2. Clarify audience, goal, and constraints if missing.
3. Draft in the requested format; offer 1 alternate tone if useful.
4. Keep the user's wording when they already provided key phrases.

## Output
- Ready-to-send draft first.
- Optional: short "why this works" only if they want critique.
