# Web Research

## When to use
Library docs, API behavior, version diffs, "what's the recommended way to…".

## Steps
1. Clarify the exact question and constraints (language, version).
2. Use available search/browse tools if present; otherwise answer from training with caveats.
3. Prefer primary docs over random blogs.
4. Summarize steps, cite links, and note uncertainty.
